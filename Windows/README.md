## 常用快捷键

  * `F2` 重命名文件

  * `Alt+F4` 关闭当前文件
  
  * `Ctrl+C` 复制光标选中的内容
  
  * `Ctrl+V` 粘贴剪切板中的内容
  
  * `Ctrl+S` 保存文件
  
  * `Alt` 焦点选中工具条
  
  * `Ctrl+Shift+N` 新建文件夹（注意：在Chrome浏览器下该快捷键是打开隐身模式）
